# 대한민국 인구전망 아틀라스 — GOALS

## 골 (Goal)

**전국 어디서든 시도→시군구로 파고들며 2031 인구 전망·소멸위험을
지도·카토그램·시뮬레이션·피라미드로 탐색하는 단일 정적 사이트.**

- GitHub Pages에 올리면 그대로 동작하는 정적 사이트 (빌드 산출물 `data/atlas.json`만 사용, 브라우저에서 API 호출 없음)
- 검증된 경북판 보고서(`kosis_gb_report`)의 뷰(팬차트·표·막대·지도·카토그램·연도 시뮬레이션·툴팁·크로스하이라이트·상세카드·하이라이트)를 전국으로 일반화
- 데이터 조작 금지: 값이 없으면 "자료 없음", 관할 변경 지역은 코드 승계·제외 규칙으로 일관 처리
